5. 自我实现————实现自己的理想

**注意事项**
1. 绝对不做免费
2. 最好不要赔本
3. 低成本不意味着便宜
4. 学会核算成本

**固定成本————耗材/实体产品**


    如何减少：
        - 合作
        - 库存
        - 使用权

## 二、赚取利润——线下成交大卡

### 1. 如何分解目标责任到人能达标，并超额完成业绩

我们常说不会分解目标的销售，基本100%是不可能成为销冠的，同样作为管理者也是没有办法做到随时随地跟进监督的。

**分解目标**

将年度目标分解到每一个季度、每一个月、每一个周、直至每一天。

**责任到人**
将团队目标分解到个人；人又分为“销售”与“客户”，“客户”又可以分为“新客户发掘”与“老客户续费升单”


### 2. 一个工具瞬间把握客户购买力和需求，轻松拿捏

在任何一次成交之前，我们对客户的购买力和需求的把握度越精准，成交的几率就会越高，这就需要我们设计以“收钱”为目的的第一次线下体验，让客户“说出”自己的购买力和需求。

**设计一张以“收钱”为目的的测试表**

客户基础信息、可以体现客户消费水平的信息、服务内容需要可以测出客户需求点、一定要以电子形式发给客户


### 3. 如何把控每个体验环节，巧妙实现提前线下成交

- 体验前，树立专业、贴心的形象，让客户想来愿意来线下体验 ；
- 体验中，精准把握客户的购买力与需求点，提高成交几率；
- 体验后，让客户深信自己的需求点可以得到改变，促进成交或裂变。


### 4. 如何用第一次线下体验让客户爱上门店掏钱付款


    销售其实就是信任传递的过程，客户愿意买单，说白了就是信任你这个人，或者是信任这个产品。

体验前：提前建立信任感，预约时，可适当加入语音环节（10秒左右），注意事项提醒
体验时：除了需求点以外，可以和客户聊一些客户正在经历的其他体验（例如化妆品或孩子的兴趣班），以赞同为主，不否定客户，多倾听，并提到和我们产品的相互配合辅助
体验后：给客户一些针对需求点的专业建议
看似不销售，其实一直在销售。。。。。


### 5. 给你全套话术模板，教你快速成交大量体验客户

当你的团队没有单独的销售部门时，你要学会一对多的体验成交方式
造势：提前在朋友圈营造氛围感，且不可提前购买
场景：需要在不要太大的空间，尽量密闭（显示出拥挤、人很多的感觉）
产品体验时，不要做成交的动作，只需要做测评，介绍我们的优势，挖掘客户的需求点，并提出改善方法
这个时候需要门店的店长，做集体成交，其他人跟单，现场签约。
提前设计一款和门店不冲突的单独的卡项，最好是组合产品，并且注明后期不满意可以退费，但是退费要设置门槛，例如不体验不可以退费，不可以无理由退费，体验感不好可以腿，我们把握主动权


### 6. 难啃的客户不要慌，零成本小成本直接逼单成交

所谓“难啃”的原因：觉得贵/不想买，所以我们要精准的把握客户不买单的原因
觉得贵：除了专业和服务以外，针对客户的需求点，给她一个爆点（可以是售价高或彰显身份的东西）
不想买：除了专业和服务以外，定制“个性化”（和本产品相关或不相关）均可


### 7. 成交时直接埋下利润产品的伏笔，超额完成业绩

我们很多销售人员在促单的时候，都会进入一个误区，多送客户几次课程或者什么赠品，我有看到过送什么杯子，雨伞的，其实这是个特别不可取的行为，我们每送去的一个东西都应该要有价值，并要带来再次购买的动作。
效果好：针对客户某一需求点，有明显的改善；
易耗品：使用频率要高，定期需要更换；
价值高：门店的售价高，塑造价值（赠送时营造稀缺感）
要回访：问客户体验的感受，需求点是否有改善（简介推销）


### 8. 利用附加价值在成交后直接裂变，赚取更多利润

客户往往在刚刚被成交买单的时候，对我们的信任度是最高的时候，往往也是最愿意做转介绍的时候。
亲情卡裂变（邀请小伙伴一起变美/锻炼/跳舞/画画。。。。。）
每一张亲情卡写上以客户名字首字母为开头的有编号的定制卡，例如LX66（数字不要以01开头）
要给客户送出去的模板，几个关键点（平时卖多少钱，因为客户的原因才会有这个送）
客户体验时，除了按照我们正常的体验流程以外，要塑造体验卡是因为老客户才有的，老客户价值感会提升，老客户会帮忙成单，从而促进成交。


## 三、全员营销——全线引爆业绩

### 1. 没做过销售的企业如何做到全员营销

#### 明确当前企业现状

1. 现金流是否稳定
2. 团队是否稳定
3. 团队成员对金钱的渴望如何
4. 领导者本身是否有销售经验
5. 团队成员是否从来对业绩没有一丝贡献
6. 公司目前每个订单能够拿出多少提成

#### 改进措施
1. 领导者熟练掌握本课程的销售知识，至少完整听课三次
2. 领导者通过学习销售知识，至少成功开单3个或以上
3. 找到团队里自己的核心支持者，让他尝试销售工作，并帮助其成功开单
4. 召开团队大会，所有人不得迟到缺席请假
    1. 当众给予开单者奖金并当众表扬，强调业绩是企业的生命线，统一思想。
    2. 公布公司对销售的提成机制，鼓励大家一起开单，为公司做贡献。
    3. 一定要在会议上鼓舞士气。
    4. 除非士气特别高涨，大家非常积极，否则此会议只统一思想，不定目标

### 2. 老师如何配合销售团队做到全员营销

6. 召开团队大会，所有人不得迟到缺席请假
    1. 领导人开会明确公司现状及未来的发展模式

    3. 公布新的提成机制
    4. 老师团队负责人发言，明确具体的工作模式

### 3. 怎么开动员会让大家知道是为自己拼

1. 与各团队负责人/核心人员先达成共识，明确这次会议的核心主旨及大概的业绩目标
2. 确定好时间、地点、人员，通知所有人不得请假迟到缺席
3. 领导先发言，明确本次活动的大概方向（暂时不讲具体方法）
4. 给每个团队5-10分钟时间讨论具体的目标及打法
5. 讨论结束后领导人根据每个团队的业绩目标进行微调
    1. 不可调整得比他们自己定的业绩过高
    2. 不可说负面的话，不可批评员工。
    3. 以正面的话为主，要专注于如何提升工作来讲
    4. 明确每个团队的具体分工与核心责任
    5. 明确具体的工作标准，哪些要做，哪些不能做
6. 进行Q&A，确保大家都明确活动的打法
7. 最后进行宣誓

### 4. 活动期间士气出现问题怎么及时调整

#### 分析出现问题的主要原因

1. 目标问题————目标含糊，员工不知道往什么方向使劲
2. 方法问题————没有工作计划，工作计划无法达成目标，工作计划偏离目标
3. 回报问题————团队没有预期回报或预期较低，个人没有预期回报或预期较低
4. 信任问题————团队之间配合出现问题、对方法不信任，对回报不信任


#### 改进措施

1. 目标问题————明确核心目标，并帮助团队对目标进行拆解，直到其知道每天怎么干
2. 方法问题————明确工作计划，每天早会夕会进行总结汇报，领导人必须在场
3. 回报问题————找到核心人员，询问其对回报的看法，如确实提成较少，适当增加奖金（非增加提成比例）
4. 信任问题————找到核心人员，询问团队配合是否有问题。是否对领导人有看法。可用聚餐、下午茶等形式让大家找回信任
