### 3. 掌握神奇的发问技巧挖掘客户真正的需求点

```
不懂发问技巧就很难和客户沟通，有时候聊着聊着就变成了“查户口”，客户难受自己也难受。这节课的主要目的就是通用马斯洛需求层次的逻辑，深挖客户的需求。
```

**马斯洛需求层次的理解**
1. 生理需求————满足基本需求，能用
2. 安全需求————保障安全稳定，没有坏的影响
3. 社交需求————友谊、爱情等亲密关系的升华
4. 尊重需求————社会上的人会因此尊重他，有了成就或名声
5. 自我实现————实现自己的理想


**根据马斯洛需求层次，层层发问进入客户的内心**
1. 启发式发问
2. 对比式发问
3. 过去式发问
4. 现在式发问
5. 未来式发问


**了解客户的核心痛点**

根据客户的回答，推敲客户真正想要的是什么。

### 5. 小卡项怎么包装客户想买而且现在就愿意买

```
是告诉学员怎么包装，还是直接说流量卡怎么包装？
```


**包装是什么意思**

包装不是吹牛、说假话、编故事，而是让客户觉得*你的东西和他相关、你的东西能解决他问题*

**为什么你的东西和他相关**

1. 提问题戳痛点
2. 我们会帮助他实现梦想
3. 你的经历让他感同身受——讲故事


**为什么你的产品能解决他的问题**

1. 成分
    1. 材料
    2. 制作流程
2. 人物
    1. 谁制作的
    2. 谁觉得好
3. 效果
    1. 用了会怎么样
4. 信任
    1. 专业的方案解答
    2. 让他放心我们能帮助他
5. 客户见证
    1. 用户评价
    2. 对比描述
    3. 视频见证
    4. 权威推荐

### 8. 附：全套成交话术模板手把手训练

#### 顾客讲价

**换位思考法** 
```
客户面对太多选择，看不到产品之间的差别，担心选错而讲价
```

- 确实现在xx产品太多，看起来都一样，价格相差这么大，你也是担心买贵了吧？
- 其实我一开始接触到xxx的时候，听到价格，是和您一样的想法，总觉得xx元太贵，xx元还差不多，你也是第一次了解到这么贵的xxx吗~

**价值体现法**
```    
从创始人初心、选品严谨、做工精细、产品效果等各个方面来体现这个产品是值得这个价格的
```

- 你让我给你便宜，其实也能便宜。为了利润我甚至可以偷工减料，但是我们不想。xx面膜需要x道工序、x道流程、经过x天后才能上市，并且保质期只有3个月。这些可以让细纹变淡，暗黄变白...

**重心转移法**
```
也可以理解为对标法
```

- 你请朋友吃顿饭有时候也要花两三百，给自己买个xx也是值得的。
- 感觉您很有爱心，您想啊，有时候您在群里发个红包都几十块，相信您不差这点钱，您放心，我安排到位。
    
**真诚发问法**
```
用一个缓和的话术，向顾客提问，了解他关于这个价格的担心和顾虑
```
     
- 我也非常想按照那个价格卖给您，但确实都是统一价格很抱歉。您担心/顾虑/在意是什么？

#### 排除对手

**攻破心理法**

- 嗯嗯，很多人会拿我们和xx对比，正常的。因为选择太多，怕选错，价格相差这么大，您肯定也不放心对吧？
- 嗯相信您第一考虑的不是价格，如果您真的是贪低价，早就买了就不会来问我了。

**产品优势法**

- 我们的产品是经过xx机构权威认证的，您可以看下，这是检测报告和证书，官方已经帮您把关过，您可以放心
- 我们的xx产品也是上过xx电视台，被xx频道多次报道过，在国人心中有一定的位置。买产品肯定买的是放心。毕竟自己用/吃进肚子里，是不能开玩笑的
- 那个产品如何我确实不太清楚，毕竟我不了解。但是我们的产品已经有x年历史，您看这个是我们创始人，自己也在用，她的皮肤/身材也非常好，xx也非常好，而且还受邀参加过xx，获得过xx大奖。这些都是效果的保障。

**引导下单类**

- 其实您还是觉得我们的产品比较靠谱，现在可以直接付定金，我马上就可以帮您安排
- 知道您已经决定好了，要买xx，您地址发给我，我来安排。是支付宝方便还是微信？可以直接转账的。
- 现在订了，3天后就到您手上，到时候你用上之后就会发现，就会知道自己的选择太对了。您是付定金还是全款呢？


#### 破解犹豫

**怕买错**
- 你现在还做不了决定，是不是怕买错了？
- 你之前是有买过类似产品，然后感觉不合适的吗？
- 我之前也有一个顾客，也是因为怕买错..... 
- 我之前也和你有过类似的想法

**想要被重视**

- 发现您特别有趣，很感谢你相信我。希望你能给我一次机会，让我帮到你，或者让我的产品真正解决你的xx问题
- 我已经把你置顶啦，你就放心吧。你这么专业我给你都不用再介绍太多了。嘿嘿还得多跟你请教一下，让我更好地服务像你这样的大咖
    
**不想被成交**
```
淡化成交痕迹
```

- 谢谢你愿意给我机会，让我给你定方案或者给建议，你也主要是想改善xx问题对吧？
    - 用发问的方式帮助顾客回忆一开始购买的动机，让他搞清楚是自己想买，而不是我们要硬推销

- 嗯嗯你需要，我刚好也专业，所以特别有信心能帮你解决/改善/满足你的xx问题/挑到适合你的产品
    
**理由不到位**

- 你看，你一开始也是因为xx问题要买这个产品，你也知道这个产品能解决你什么问题，所以肯定要满足自己呀！
- 我也知道你要改善的是xx问题，刚好我们的产品就能解决，你肯定不会拒绝让自己的问题消失，所以买的是好心情。
    
**后续麻烦大**

- 确实市场上有一些产品会出现xx的麻烦，就是因为有新闻报道，我们才会更加注意，也去避免这些问题。你放心吧。不好的我不会推荐给你的。

#### 洞察客户
    搞不懂客户在搞啥，对话不好进行
    
**以诚相待法**

- 您好，我感觉自己挺专业，是不是我没讲解到位或者哪里没讲好，你不回我哈？
- 我真觉得我们x挺好，感觉你也很想买，但是我不知道你真的担心什么？方便和我说下吗？

**强化问题法**

- 你一开始也是因为xx所以加我微信的对吧？
- 你现在也主要是因为xx，导致你xx，你也想xx对吧？

**目标启发法**

- 你一开始来咨询，也是希望能xx对吧？
- 很多人买xx都是因为xx，达到xx效果，你肯定也可以/也希望对吧？

**对话复盘法**

- 好的，我们也聊了挺久，我来确认下，你是因为xx，要买xx，希望达到xx效果是吗？
- 嗯，谢谢你相信我和我说了这么多，你也是希望能买到xx产品对吧？但还是因为xx不放心是不是？
#### 客户追销

**有意提醒**

- 直接引导+利益点
    - 下次你朋友有需要的也可以帮我们介绍，只要报你名字，我们一定给你一样的优惠
    - 下次你和朋友一起报，我还能给你送xxx
- 心理暗示+理由
    - 你用好了一定会再买的，我们家老顾客就是这么来的
**无意提醒**
- 启发引导+机会暗示
    - 你们一定要好好做业绩，帮我把老板请我的培训费挣回来。也希望下次再见到大家
- 复购暗示+自信
    - 我们很多老顾客都是xx xx的买，这都是质量和效果的体现，你放心吧
- 第三方启发+利益点
    - 上次有个顾客，介绍了3个朋友过来买，我直接给她送了个xxx
